Ataque de Força Bruta (Brute Force Attack)

Desafio focado em estudar ataques de força bruta, utilizando Kali Linux e ferramentas de teste em serviços vulneráveis.

Sugestões do que incluir neste desafio:

Montagem de um laboratório com duas VMs (ex.: Kali Linux + máquina vulnerável como Metasploitable ou DVWA) no modo only-host.

Uso de uma ferramenta de força bruta (por exemplo, Medusa) para testar credenciais em serviços como FTP ou formulários web.

Registro dos comandos utilizados, resultados obtidos e reflexões sobre medidas de mitigação (senhas fortes, bloqueio por tentativas, etc.).

Pasta sugerida: brute-force-attack/
